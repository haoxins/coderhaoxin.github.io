### 测量一个指令的试行时间

```go
times := 1_000_000_000
start := float64(time.Now().Unix())
for i := 0; i <= times; i++ {
  // 指令
}
end := float64(time.Now().Unix())
fmt.Print((end - start) / float64(times))
```

- 本质上, 原子等微观态的测量类似
