  - 至于所谓的前沿, 也没啥展开描述, 所以也就剩下个代码部分了~

```
执行不可逆运算就会丢失信息, 因为这相当于对系统的状态进行了测量.
这样一个计算周期便完成了, 程序无法继续执行.
相反, 如果使用可逆逻辑门, 只要保持系统的相干性, 就可以继续将算子作用于量子比特.
```

> 不可逆运算, 相当于测量~

```
只有当所考虑的向量的项全为实数时, 点积和内积才是一致的. 
```
